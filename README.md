## 📌 Project Overview
This project implements an advanced version of the classic Rock-Paper-Scissors game using SWI-Prolog. Unlike the traditional random gameplay, this system introduces an intelligent opponent that makes decisions based on logical rules and past user behavior.

The project demonstrates core AI concepts such as knowledge representation and reasoning using Prolog’s rule-based system.

---

## 🚀 Key Highlights
- Implementation using **facts, rules, and recursion**
- Use of **backtracking** for decision-making
- Dynamic knowledge base that updates during gameplay
- AI opponent that adapts based on player patterns
- Fully interactive command-line interface

---

## ⚙️ Features
-  Standard gameplay with proper input validation  
-  Smart opponent that learns from previous moves  
-  Dynamic memory using Prolog predicates (`assert` and `retract`)  
-  Multiple difficulty levels for varied gameplay
