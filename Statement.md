#  Project Statement

##  Problem Description
The traditional Rock-Paper-Scissors (RPS) game relies purely on random choices, making it predictable over time. This project aims to design a smarter opponent by using logic-based decision-making instead of randomness.

---

## Project Scope
The goal is to develop a complete RPS game using SWI-Prolog that incorporates an intelligent agent. The system uses rule-based logic and dynamic knowledge representation to analyze player behavior and adapt strategies over time. This aligns with key concepts of Prolog programming and knowledge representation in AI.

---

##  Target Users
- Students studying Artificial Intelligence.  
- Beginners learning Prolog and logic programming.  
- Anyone interested in simple AI-based games.  

---

##  Key Features
1. Command-line based interactive gameplay with proper input handling.  
2. Intelligent opponent using rule-based logic.  
3. Learning mechanism that updates knowledge from previous moves.  
4. Adaptive strategy using Prolog facts, rules, and backtracking.
